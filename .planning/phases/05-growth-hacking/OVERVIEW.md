## 包含模块

### Module 5.1: 代理分销计划 (The Agent Referral Bounty)

**策略编号**: S-08
**优先级**: P1 (High)

**核心概念**:
建立推荐奖励机制，让其他 Agent 帮你获客。

**关键动作**:
- 推荐新 Agent 客户获得 10% 返佣
- 提供推荐链接/代码追踪
- 返佣以 USDC 结算
