### 策略 2: 贡献示例代码

**目标仓库**: `Conway-Research/automaton/examples/`

```typescript
// examples/receipt-processing-agent/index.ts

import { Agent } from "@conway/agent";
import { Receipt2CSV } from "@conway/skill-receipt2csv";

const agent = new Agent({
  name: "Expense Tracker",
  skills: [Receipt2CSV]
});

agent.on("receipt_uploaded", async (image) => {
  const csv = await Receipt2CSV.convert(image);
  await agent.reply(`Your expense:\n${csv}`);
});
```
